
# Smart Waste Collection Optimization 🚛

## 📌 Overview
This project uses Machine Learning to predict which waste bins need collection.

## 🔥 Features
- Random Forest ML model
- Confusion Matrix evaluation
- Feature importance analysis
- Smart priority-based collection
- Map visualization using Folium

## 📊 Results
- Accuracy: ~85-90%
- Realistic predictions using probabilistic data

## ▶️ How to Run
pip install -r requirements.txt
python src/train.py
python src/predict.py
